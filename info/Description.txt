[font=Verdana][size=4][b]Description[/b][/size][/font]
The mod adds a text widget to the HUD that displays the current ping latency to the server and allows you to adjust the alignment on the screen, the size and the color of the text via the mod options.
You can also specify whether it should only be visible in the tactical overlay. (Overlay that by default can be opened with the Tab key)
This mod is an alternative to [url=https://www.nexusmods.com/warhammer40kdarktide/mods/13]Ping Monitor[/url].

[font=Verdana][size=4][b]Requirements[/b][/size][/font]
[size=3]- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url]
- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url][/size]

[font=Verdana][size=4][b]Installation[/b][/size][/font]
1. Read descriptions and install [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url] and [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url]
2. In [b]Files[/b] tab use the [b]Mod Manager Download[/b] button to install the mod with [url=https://www.nexusmods.com/about/vortex/]Vortex[/url].
3. Make sure the mod is enabled in Vortex's [b]Load Order[/b] tab.
4. Done.

[font=Verdana][size=4][b]Differences to the Ping Monitor mod[/b][/size][/font]
These are the key differences to the existing ping display mod [url=https://www.nexusmods.com/warhammer40kdarktide/mods/13]Ping Monitor[/url].
[list]
[*]Display Ping has options to adjust the text alignment on the screen with correction via offset values, customizable color and text size.
[*]An option to add a customizable Label on the left or right side of the Ping value.
[*]Ping Monitor shows the ping only in the "Tactical Overlay", while Display Ping gives you the option to show the ping all the time.
[*]Ping Monitor shows average ping of last 30sec, Display Ping has the option to show average, but per default it shows latest measured latency.
[*]Display Ping uses current ping value for Dynamic Color Range, Ping Monitor shows the text in orange or red depending on jitter.
[*][url=https://www.nexusmods.com/warhammer40kdarktide/mods/10]Custom HUD[/url] support.
[/list][font=Verdana][size=4][b]Feedback[/b][/size][/font]

[font=Verdana][size=4][b]Report Technical issues[/b][/size][/font]
Please report any bugs in comments or create a "Bug Report".
However, if you encounter technical issues like an error, game crash, the Ping doesn't show up on the HUD, please include the last game log.
Logs can be found under: [i][b]%appdata%\Fatshark\Darktide\console_logs\[/b][/i]
You can upload the content of the file to [url=https://pastebin.com/]pastebin.com[/url] or a similar site.
